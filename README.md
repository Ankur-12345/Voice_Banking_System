# Voice_Banking_System
AI automated voice banking system

1. Create a virtual environment by "python .venv venv" and activate it ".\venv\scripts\activate"
2. Install all dependencies from requirements.txt //pip install -r requirements.txt
3. Install all the required nodes "npm install"
4. Run the backend server after going to it's directory "cd backend" and run it by "uvicorn app.main:app --reload"
5. Run the frontend server on another terminal after going to it's directory "cd frontend" and run by "npm start"





Enjoy Voice Banking!
