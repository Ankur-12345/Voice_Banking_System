# Voice_Banking_System
AI automated voice banking system

Create a virtual environment by "python .venv venv" and activate it ".\venv\scripts\activate"
Install all dependencies from requirements.txt //pip install -r requirements.txt
Install all the required nodes "npm install"
Run the backend server after going to it's directory "cd backend" and run it by "uvicorn app.main:app --reload"
Run the frontend server on another terminal "npm start"

Enjoy Voice Banking!
